# Library Management System

## Overview
A desktop-based Library Management System developed using Java Swing and MySQL to automate library operations.

## Features

- User authentication
- Manage books
- Manage authors
- Manage members
- Issue and return books
- Search books
- Overdue tracking
- Dashboard

## Technologies

- Java
- Java Swing
- MySQL
- JDBC
- NetBeans

## Database

MySQL

## Installation

1. Clone this repository
2. Open the project in NetBeans
3. Import the MySQL database
4. Configure database connection
5. Run the application
